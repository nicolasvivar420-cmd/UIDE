# Adivina el Número

## Fecha

## Objetivo del sistema
Desarrollar un programa en Python que adivine el número que el usuario 
está pensando, utilizando el método de búsqueda binaria para encontrarlo 
en el menor número de intentos posible.

## Descripción del problema
Muchas veces queremos demostrar cómo un algoritmo puede ser más eficiente 
que una búsqueda aleatoria. Este programa resuelve ese problema adivinando 
cualquier número del 1 al 100 en máximo 7 intentos, gracias a la lógica 
de búsqueda binaria.

## Funcionalidades
- El usuario piensa un número entre 1 y 100
- El programa pregunta si su número es mayor, menor o igual al que propone
- Usa búsqueda binaria para reducir las posibilidades en cada intento
- Cuenta los intentos hasta adivinar el número
- Detecta si el usuario ingresa respuestas contradictorias
