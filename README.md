# Adivina el Número

## Fecha

## Objetivo del sistema
Desarrollar un programa en Python que adivine el número que el usuario 
está pensando, utilizando el método de búsqueda binaria para encontrarlo 
en el menor número de intentos posible.

## Descripción del problema
En el aprendizaje de programación, uno de los retos principales es 
entender cómo los algoritmos pueden resolver problemas de forma eficiente. 
Este proyecto demuestra cómo la búsqueda binaria permite adivinar cualquier 
número del 1 al 100 en máximo 7 intentos, siendo mucho más eficiente que 
una búsqueda aleatoria.

## Funcionalidades
- El usuario piensa un número entre 1 y 100
- El programa pregunta si su número es mayor, menor o igual al que propone
- Usa búsqueda binaria para reducir las posibilidades en cada intento
- Cuenta los intentos hasta adivinar el número
- Detecta si el usuario ingresa respuestas contradictorias

## Explicación del sistema
El programa utiliza tres variables principales: mínimo, máximo e intentos. 
En cada turno calcula el punto medio del rango disponible y le pregunta al 
usuario si su número es mayor, menor o igual. Según la respuesta, ajusta 
el rango y repite el proceso hasta adivinar el número exacto.

que puede resolver problemas reales de forma eficiente. La búsqueda binaria

hacer en 7 pasos lo que una persona haría en 50. Como futuro ingeniero en IA,
